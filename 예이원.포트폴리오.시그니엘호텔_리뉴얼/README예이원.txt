---------------------------------------------------  페이지들 소개

- 1번째 페이지 소개 
	왼쪽 - 객실로 감 
      	오른쪽 - 호텔 메인페이지 
    
      	이렇게 나눈 이유 : 
      	객실만 구경하고싶어하는 사람을 위한 간편하게 이동하게하기 위함 
	
	스크립트 기능 없음 
	처음에 애니메이션으로 글씨가 내려옴 
	페이지 이동버튼 호버 시 백그라운드로 색상 채워짐 

- 2번째 페이지 소개  
	
	만든 의도 그저 색다름을 주고싶었을 뿐 만들고나서 보니 왜 만들었나싶음 
	에러가 살짝 남 

	기능 -
	1. 스크롤링하면 가로로 풀페이지 이동
	2. 헤더 햄버거 버튼(css로함) 
	3. 햄버거 버튼 클릭 시 전체화면으로 메뉴 내려옴 

----------------------------------------------------- 메인 페이지 
	

-3번째 페이지 소개 (메인) 어느 페이지에서도 로고를 누르면 넘어감 

	주로 전체적인 소개를 보이게끔 했고 위치와 기본적인 소개글 상단 메뉴와 
	3번쨰 섹션와 상단 메뉴바로 반응형 시에도 편리함을 위해 기능을 넣었음 

	기능 - 
	1. 풀페이지 스크롤링 한페이지씩 내려감 
	2. 메인배너 이미지 슬라이드 호버 시 버튼 있음 
	3. 상단메뉴 호버 시 로고 색상과 백그라운드 색상 변경
	4. 상단메뉴 호버 시 메뉴바 내려옴 (이 기능에 대한 세심한 기능은 추가하지 못함) 
	5. 푸터 = 1,2번쨰 페이지 제외 항상 있을 것 아이콘마다 링크태그 달았음 

---------------------------------------------------  상세 페이지라 특별한 기능은 전체적으로 많지 않음 

전체적으로 상단 메뉴바가 안전하게 잘 내려오는 것만 신경썼음 

-4번쨰 페이지 소개 
	객실리스트목록
	레이아웃에 중점을 두었음  

	2번쨰 페이지를 보지 않아도 간편하게 전체적인 객실들을 한 눈에 보기 편하기 위함으로 만듦 
	특별한 기능은 없음 

	사진 모두에게 슬라이드를 주면서 객실의 사진을 슬라이드로 넘기면서 볼 수 있게 하려 했으나 능력의 한계

-5번째 페이지 소개 
	다이닝 목록 
	각각의 상자를 클릭하면 그 식당으로 넘어가는 컨셉으로
	어느 곳을 클릭하는 지 알아보기 편하도록 호버 시에 하얀 선만 있는 박스와 하얀 글씨로 나타내줌 

-6번째 페이지 소개
